Installation

Three small steps.

The installer handles the plugin, maps, and Viewer together.

  1. 01

    Get your installer

    Choose the installer for your operating system above. It downloads directly from the latest GitHub release.

  2. 02

    Run it

    Select your VPX installation and tables folder when prompted. The installer places everything where it belongs.

  3. 03

    Finish a game

    Play a supported PinMAME table through game over. Score Tracker records the result automatically.

Linux: make the downloaded installer executable before opening it: chmod +x scoretracker-installer-linux-*

VPinballX.ini

One setting matters.

The installer enables Score Tracker automatically. For a manual installation, find or add this section in VPinballX.ini.

VPinballX.ini Required 
[Plugin.ScoreTracker]
Enable = 1

Restart VPX after changing the file.
